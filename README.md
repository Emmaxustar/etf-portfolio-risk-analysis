# ETF Portfolio Risk Analysis in R

This project analyzes a five-ETF portfolio using monthly return data. The ETFs include SPYM, VEA, VWO, JCPB, and SPSM.

The goal of this project is to evaluate portfolio risk-return performance, diversification benefits, and asset allocation decisions using R.

## Project Overview

This project covers:

- Monthly return calculations and return statistics
- Sharpe ratio analysis
- Normal and empirical Value-at-Risk
- Rolling mean, rolling volatility, and rolling correlation
- Global minimum variance portfolios
- Efficient frontier analysis
- Tangency portfolio analysis
- Risk budgeting
- Target-return asset allocation

## Tools Used

- R
- R Markdown
- IntroCompFinR
- PerformanceAnalytics
- ggplot2
- xts / zoo
